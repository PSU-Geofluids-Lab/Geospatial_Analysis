SIS India applications

Start year
2018
Share on:
End year
2021

The 'Building a Soil Intelligence System (SIS) for India project' considers three states, Andhra Pradesh, Bihar and Odisha. The system will support better decision-making in agriculture. This page presents two outputs of the project. The first is a web viewer with access to the gridded soil maps that were developed for the project. The second is a prototype dashboard that gives examples of how soil information, aggregated to two administrative levels, could be presented to the end user. The dashboard obtains its information from a database system that contains the data from the Soil Health Card initiative.
